# 5 Day Weather App

#### By **Nanette Girzi**

## Description

Basic React/Redux application that shows the 5 day forecast (temp, pressure and humidity) for cities entered by the user.

## Setup/Installation Requirements

Run the following commands in Terminal:

* $ git clone
* $ cd WeatherApp
* $ npm install
* $ npm run start
